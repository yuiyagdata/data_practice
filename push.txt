- ローカルレポジトリを指定
	- mkdir github
	- cd github
	- mkdir pushtest
	- cd pushtest
	- git init
- ローカルレポジトリからリモートレポジトリへpush
	- git add ファイル名
	- git commit -m “pushコマンドの確認”
	- git remote set-url origin https://github.com/yuiyagdata/レポジトリ名.git
	//git remote -v で確認すると、新しいリポジトリURLになってるはず！
	- git push -u origin master
